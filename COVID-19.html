Introduction

The COVID-19 pandemic has and continues to devastate the world economically and socially, causing everyone's every way of life to change drastically. Like many other students, my classes were held virtually for the past year due to the virus.

There are many complaints from students and teachers stating how the pandemic has negatively impacted academic performance. Today, I want to investigate whether of not this is statistically true: was there a significant drop in academic performance during the COVID-19 pandemic.

Why are we doing this? COVID-19 has devastated many students academically, physically, and mentally. Being able to identify specific traits or variables that exacerbate these devastations will allow appropriate government/group action to combat these issues.

First we need to import some libraries that will be useful for our analysis. Information for the important libraries can be found at the bottom of the page.

Data Collection

Who's the test group?

Due to the time restraint of this course and number of students, I am unfortunately unable to obtain the data from every single student. Therefore, I restricted my population to K-12 students in the state of Maryland. Along with this, I collected data from each county instead of a cumulative measurement. This is because the circumstances in each county are vastly different.

How do we define academic performance?

There's no one perfect way of measuring academic performance as the term is very general. Normally, I would obtain test results from the MAP test, a state-wide assessment given to every student anually. However, due to the pandemic, standardized tests were cancelled for 2020/2021, making this option unavailable.

Instead, I obtained data on the number of students failing a course. More people failing indicates overall worsened academic performance and vice-versa.

What data are we using?

Sadly, there is no one dataset that contains everything we need. However, using the power of DataFrames, we can concatenate data from multiple sources. There are 3 main pieces of information I need: number of students failing a course, COVID cases, and demographic data.

Students Failing a Course
The number of student failing was obtained from a report from the Maryland Department of Education. This document was published here by WJZ, a local news station. I converted the table into a .csv file and imported it to jupyter.

Covid Cases by County
The number of COVID cases was obtained from a public database published by the state of Maryland. The specific table can be found here. The information was able to be exported as a .csv file beautifully.

Demographics by County
To gain more insight into the data for any trends, I also wanted to obtain demographic data for each county. I found a table online that provides population information for each county The specific table can be found here. The information was able to be exported as a .csv file.

Another table was found here here . This contains information such as the number of students per county, and ethnicity breakdowns. This was published by the Maryland Department of Education. I converted the table into a .csv file and imported it to jupyter.

An additional table was found here here from the Maryland Department of Commerce. We use this table to obtain the income per capita

    # edit county names for population
def edit_names_population(row):
    # remove county
    row.CTYNAME = row.CTYNAME.replace('County', '')
    # remove period
    row.CTYNAME = row.CTYNAME.replace(' ', '')
    # remove ''
    row.CTYNAME = row.CTYNAME.replace('\'', '')
    # remove .
    row.CTYNAME = row.CTYNAME.replace('.', '')
    return row.CTYNAME

# edit county names for enrollment 
def edit_names_enrollment(row):
    # remove period
    row.Local_School_System = row.Local_School_System.replace(' ', '')
    # remove ''
    row.Local_School_System = row.Local_School_System.replace('\'', '')
    # remove .
    row.Local_School_System = row.Local_School_System.replace('.', '')
    row.Local_School_System = row.Local_School_System.replace('Saint', 'St')
    # remove , in order to cast to float
    row.Total_Students = row.Total_Students.replace(',', '')
    return row.Local_School_System

    Hypothesis Testing

    Let's first determine if there is a significant increase in number of students failing a course between 2020 and 2021. In order to compare the values between counties, we must used the normalized values.

    We will be using a paired t-test to compare the failure rates. The null hypothesis is that there is no differece in failure rates between the two years. The alternative hypothesis is that there is a difference in failure rates between the two years.

    In [41]:
    y = school_stats_2021['Failed/Students'].tolist()
y_2020 = school_stats_2021['Failed_2020/Students'].tolist()
stats.ttest_ind(y_2020, y)
    Out[41]:
    Ttest_indResult(statistic=-4.2792395680335495, pvalue=9.403797271395658e-05)

    The p-value is much less than 0.05. Therefore we can reject the null hypothesis and conclude that there is significant difference in failure rates between 2020 and 2021.

    To see if there is a significant correlation between the number of students failing a course and COVID cases. We can use the Pearson correlation coefficient. It gives us a value between -1 and 1 which expresses the degree in which the two variables change. We calculated this previously already. The important value is the p-value given.

    The null hypothesis is that the two variables are independent (i.e. one changeing in value does not affect the other). The alternative hypothesis is that there is some dependency between the variables. If the p-value is less than 0.05, we can reject the null hypothesis.

    Insight and Conclusions

    There were many insightful trends that we found in our analysis.

    Significant Increase in Failure Rate


    From the grouped vertical bar graph, we saw that for every Maryland county (besides Howard), there was a significant increase in failure rates. In our analysis, we found that there was a significant increase in the failure rates between 2020 and 2021 using a paired t-test. This points to the idea that something must have occurred between these two years to cause this change.

    Linear Correlation Between Failure Rates and COVID Cases/Population


    From our scatter plot, we found a moderate positive correlation between these two variables. After calculating the Pearson correlation coefficient, we were able to conclude that there is a linear correlation between the failure rate and COVID cases/population. As COVID cases increase, the failure rate of student also increases. This makes sense as the virus can physically and mentally damage an individual and people around them. If a student or a close person to a student was infected. They would have to priortize their health over the academic duties. However, future studies would need to be conducted to validate these speculations

    Linear Correlation Between Failure Rates Change and Income Per Capita


    Looking at the group vertical bar graph, I noticed that a the biggest increases in failure rates were found in counties with low incomes per capita. After making a scatter plot, we found a moderate negative correlation. After calculating the Pearson correlation coefficient, we were able to conclude that there is a negative linear correlation between the failure rate change and income per capita. As the income per capita increase, the failure rate of student decreases. This makes total sense as wealthier familiies are able to buy resources that allow their children (students) to be more prepared to school, leading to lower failure rates. However, this correlation shows that wealthier families were less impacted academically between two years. This might be because wealthier families have much more disposable income are able to prioritize their health/academic duties unlike impoverished families that have to continue working. It's also worth noting that higher paying jobs are able to be performed at home virtually while many lower paying jobs like the food service industry must be performed in-person. This in turn increases the chance to be exposed to the virus for people working in these jobs, thus placing more stress on their families. Future studies would need to be conducted to validate these speculations
